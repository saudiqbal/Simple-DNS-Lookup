# Simple-DNS-Lookup
PHP DNS Lookup script

Simple PHP script to lookup entries of A, AAAA, CNAME, NS, MX, SOA and TXT records.

You can test the tool here : https://outils.hqweb.com/dns-lookup/

Removed external api.

Build on Bootstrap.
